# Foundry Labs Landing

## Stack
- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS 3

## Package Manager
- **Use `bun`** for all package management and script execution (`bun install`, `bun run dev`, `bun run build`, etc.)

## Commands
- `bun run dev` — start dev server
- `bun run build` — production build
- `bun run start` — start production server
- `bun run lint` — run linter

## Design Context

### Users
Founders and CTOs evaluating senior engineering partners for AI-driven products. They arrive skeptical, time-constrained, and looking for signals of real expertise. The site must convert high-intent visitors into Calendly bookings. Secondary audience: LATAM Spanish-speaking founders.

### Brand Personality
**Bold, Direct, Modern** — No hedging, no filler. The site speaks with the confidence of senior engineers who've shipped.

### Emotional Goal
**Confidence & calm** — "These people clearly know what they're doing — I can relax."

### Aesthetic Direction
- Premium product feel with rich interactions — inspired by Stripe and Raycast
- Editorial typography (Playfair serif headings, Inter sans body)
- Warm paper surface (#F5F7FA), ink text (#111), purple-to-pink gradient accents
- Subtle sophisticated motion, always respecting `prefers-reduced-motion`
- Anti-references: generic agency sites, stock photos, buzzword-heavy copy

### Design Principles
1. **Earned presence** — Every element must justify its existence
2. **Quiet confidence** — Premium is felt in spacing, typography, and restraint
3. **Interaction as proof** — Micro-interactions signal engineering quality
4. **Clarity over cleverness** — Immediately understood, no jargon gates
5. **Accessible by default** — Current a11y patterns are the floor, not the ceiling
